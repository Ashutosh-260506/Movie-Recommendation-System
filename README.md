# 🎬 Movie Recommender System

A **Content-Based Movie Recommender System** built using **Python**, **Scikit-Learn**, and **Streamlit**. The application recommends the top 5 movies similar to the selected movie using cosine similarity.

## 🚀 Features

- Recommend top 5 similar movies
- Content-Based Filtering
- Interactive Streamlit UI
- Dark Theme Interface
- Fast recommendations using a precomputed similarity matrix

## 📊 Dataset

- **TMDB 5000 Movie Dataset**
- Source: https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
